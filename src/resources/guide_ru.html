Симулятор тоннеля - Руководство пользователя

1 Назначение системы

Автоматизированная система моделирования движения транспорта в тоннеле предназначена для симуляции движения автомобилей в тоннеле, визуализации процесса движения и управления им.

2 Установка системы

Для запуска системы установка не требуется, достаточно поместить JAR файл системы в каталог не защищённый от записи, сохранив оригинальное имя JAR файла.

3 Требования к программным и аппаратным средствам

  1. Тип ЭВМ - IBM PC совместимый;
  2. 18,001 Гб на жестком диске;
  3. 1302 Мб оперативной памяти;
  4. тип операционной системы - любая поддерживающая Java Virtual Machine Standard Edition версий 7 и выше;
  5. Монитор с разрешающей способностью не ниже 1024х768;
  6. Манипулятор – мышь;
  7. Клавиатура.

4 Описание основных действий пользователя

1. Запуск приложения

Запуск приложения сопровождается заставкой. Продолжительность заставки 6 секунд. По прошествии 6 секунд заставка исчезает и появляется основная форма программы.

2. Выбор языка системы

Выбор языка осуществляется в меню Language/Язык.

3. Старт симуляции

Инициализировать процесс симуляции можно 2 способами:
  1. вручную, нажатием на кнопку Следующий шаг;
  2. автоматически, включив режим автосимуляции;

4. Задание потока появления машин

Выбрать поток появления машин можно зайдя меню поток машин, после чего будет применён выбранный поток и появится форма его настройки.

5. Настройка длины дороги и масштаба

Настройка длины дороги и масштаба осуществляется в форме Параметры дороги, при этом стоит учитывать, что при изменении длины дороги процесс симуляции начинается сначала. Параметр увеличение характеризует скольким пикселям на экране соответствует один метр модели.

6. Ручное управление скоростью автомобиля

Система позволяет вручную регулировать скорость автомобиля, для этого предназначена форма Ручное управление. Выбранный автомобиль выделяется на форме визуализации состояния модели более толстой линией, а также красным цветом.

7. Создание и удаление моделей машин

Если необходимо создать или удалить модели машин, то это можно сделать зайдя в меню Машины: Показать модели. В результате отобразится форма для создания и удаления моделей машин.
Для того чтобы создать новую модель, вводим её название, максимальную скорость и цвет(задаётся компонентами R(red), G(green), B(blue)), затем нажимаем на кнопку Добавить модель.
Чтобы удалить модель, выделяем нужную строку и жмём кнопку удалить.

8. Сохранение и загрузка моделей машин

Для загрузки/сохранения списка моделей заходим в пункт меню Машины. Результатом нажатия на Загрузить модели будет диалоговое окно позволяющее осуществлять навигацию по файловой системе используемой ОС.
Для загрузки списка моделей выделяем нужный файл(с расширением .cars) и жмём Загрузить.
Процесс сохранения аналогичен.

5 Горячие клавиши системы

  • Пробел (Space) - следующий шаг симуляции;
  • Стрелка вправо (Right arrow key) - следующая машина;
  • Стрелка влево (Left arrow key) - предыдущая машина;
  • Стрелка вверх (Up arrow key) - увеличить скорость текущей машины на 1 м/с;
  • Стрелка вниз (Down arrow key) - уменьшить скорость текущей машины на 1 м/с.
